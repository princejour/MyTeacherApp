# Release Checklist

Use this checklist before every release.

## App Identity

- [ ] App name confirmed
- [ ] Package name confirmed
- [ ] App icon added
- [ ] Splash screen added
- [ ] Version code increased
- [ ] Version name updated

## Firebase

- [ ] Production Firebase project ready
- [ ] Android app registered in Firebase
- [ ] google-services.json added locally or through secure workflow
- [ ] Firestore rules reviewed
- [ ] Cloud Messaging tested

## AdMob

- [ ] Test rewarded ads working
- [ ] Production rewarded ad unit created
- [ ] Production rewarded ad unit inserted
- [ ] Ads tested without policy violations

## Privacy and Store

- [ ] Privacy policy final
- [ ] Terms of use final
- [ ] Support email added
- [ ] Store listing completed
- [ ] Data Safety completed
- [ ] Screenshots ready

## Build

- [ ] Signing key configured
- [ ] GitHub Secrets configured
- [ ] Release APK or AAB generated
- [ ] Installation tested
- [ ] Update over previous version tested

## Final Review

- [ ] Teacher flow tested
- [ ] Family flow tested
- [ ] Message limit tested
- [ ] Rewarded unlock tested
- [ ] No test credentials exposed
