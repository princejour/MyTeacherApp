# Data Safety Notes

These notes are a preparation checklist for the app store data safety form.

## Data Types to Review

- Names added by the teacher
- Class information
- Private family access codes
- Messages
- Message unlock status
- Monthly access counters
- Device notification tokens
- Ad interaction status for rewarded unlocks

## Services to Review

- Firebase Firestore
- Firebase Cloud Messaging
- Google AdMob Rewarded Ads

## Required Before Release

- Confirm exactly what data is collected.
- Confirm why each data type is needed.
- Confirm whether data is shared with third-party services.
- Confirm whether data is encrypted in transit.
- Confirm whether users can request deletion.
- Make the Data Safety answers match the privacy policy.

## Notes

The app should not collect unnecessary information. Only data required for app functionality should be used.
