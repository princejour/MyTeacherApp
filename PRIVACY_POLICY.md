# Privacy Policy

Effective date: To be updated before release.

My Teacher is a communication app for teachers and parents.

## Data Used by the App

The app may use:

- Student names
- Class information
- Parent access codes
- Teacher messages
- Message unlock status
- Monthly message counters
- Device notification tokens

## Purpose

This data is used to:

- Show parent messages
- Manage student lists
- Send notifications
- Apply the monthly free message limit
- Unlock extra messages after rewarded ads
- Keep the app working securely

## Firebase

The app uses Firebase services for database storage and notifications. Production setup must use secure Firestore rules and real Firebase configuration.

## AdMob

The app may use Google AdMob Rewarded Ads. Rewarded ads are used to unlock additional messages after the monthly free limit.

## Data Sharing

The app does not sell user data. Data may be processed by required service providers such as Firebase and AdMob.

## Data Retention

Data is kept while it is needed for app functionality. Authorized deletion requests should be handled by the app owner or school administrator.

## Security

Production release must protect credentials, restrict database access, and keep private configuration files outside the public repository.
