## What to Include in a Support Request

Please include:

- App version
- Android version
- Device model
- Screenshot of the issue if possible
- Steps to reproduce the issue
- Whether the issue is in teacher mode or parent mode

## Common Issues

### Parent code does not work

Check that the code was copied exactly and that the student still exists in the active class list.

### Messages do not appear

Check the internet connection and Firebase configuration.

### Rewarded ad does not load

Check AdMob setup, internet connection, and whether test ads or production ads are being used.

### Teacher password does not work

Use the current password or reset the app configuration according to the release policy.
