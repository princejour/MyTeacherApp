# FAQ

## What is My Teacher?

My Teacher is an Android app designed to help teachers send secure messages to parents through student-specific access codes.

## Who can use the app?

The app is designed for two main users:

- Teachers who manage students and send messages.
- Parents who open messages related to their child.

## How does the parent access system work?

Each student receives a unique parent access code. Parents use this code to open the message inbox linked to their child.

## How many messages can a parent open for free?

Parents can open 5 messages for free each month.

## What happens from the 6th message?

Starting from the 6th message in the same month, the message stays locked until the parent watches a rewarded ad.

## Does the message unlock immediately when the ad starts?

No. The message is unlocked only after the rewarded ad is completed successfully.

## Can the teacher change the password?

Yes. The teacher can change the password from inside the app.

## Does the app need Firebase?

Yes. Firebase is used for student data, parent access codes, messages, notifications, and usage tracking.

## Does the app use AdMob?

Yes. The planned premium model uses AdMob Rewarded Ads to unlock messages after the monthly free limit.

## Is this repository ready for production?

This repository is being prepared for a premium English release. Production release requires complete Android source code, Firebase setup, AdMob setup, signing configuration, privacy policy, and Google Play Data Safety declarations.
